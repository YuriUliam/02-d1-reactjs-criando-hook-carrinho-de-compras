# Módulo II - Desafio 1 - Criando um hook de carrinho de compras

Essa é uma aplicação onde o principal objetivo é criar um hook de carrinho de compras. Tive acesso a duas páginas, um componente e um hook para implementar as funcionalidades pedidas nesse desafio:

- Adicionar um novo produto ao carrinho;
- Remover um produto do carrinho;
- Alterar a quantidade de um produto no carrinho;
- Cálculo dos preços sub-total e total do carrinho;
- Validação de estoque;
- Exibição de mensagens de erro;
- Entre outros.
